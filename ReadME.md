# NixieTubeAudio
此為課程 系統晶片的課堂專題，使用RX62T控制的輝光管收音機

於課堂結束後，因RX62T控制板繳回，改用arduino uno重新撰寫，並使用快速傅立葉轉換系統音訊，透過UART控制RGB LED
